# 1. R 설치 및 기본 사용법

## (2) RStudio를 활용한 R 기본 사용법

### ② R의 데이터 타입

#### ⓐ 문자형 타입

```
class('abc')

class("abc")

class('1')

class("TRUE")
```

#### ⓑ 숫자형 타입

```
class(Inf)

class(1)

class(-3)
```

#### ⓒ 논리형 타입

```
class(TRUE)

class(FALSE)
```

#### ⓓ NaN, NA, NULL

```
sqrt(-3)

class(NA)

class(NULL)
```
